Digit Recog

Digit Recog is a web application that uses Tensorflow.js
to predict a user-drawn digit to the best of it's ability.
The TensorFlow model was trained against 55000 data
entries from the MNIST database in order to increase
the probability of correctly predicting the user-drawn
image.

This project was made in collaboration with Farhan,
Sanjeev, and Ashanth for Hack the Valley 4.
